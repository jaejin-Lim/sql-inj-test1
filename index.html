SQL 인젝션이란?

SQL 인젝션(SQL Injection)은 공격자가 애플리케이션의 입력 필드에 악의적인 SQL 코드를 삽입하여, 데이터베이스 쿼리를 조작하는 공격 기법입니다.

개발자가 의도하지 않은 쿼리가 실행되면서 인증 우회, 데이터 탈취, 데이터 삭제 등의 피해가 발생합니다.

정상 쿼리

SELECT * FROM users WHERE username = 'admin' AND password = '1234'

인젝션 후 쿼리

SELECT * FROM users WHERE username = '' OR '1'='1' --' AND password = ''

왜 위험한가?

  • 🔓
    인증 우회

    올바른 비밀번호 없이도 로그인 가능

  • 🗂
    데이터 탈취

    전체 데이터베이스의 내용 열람 가능

  • 💥
    데이터 파괴

    DROP TABLE 등으로 데이터 삭제 가능

  • 🔑
    권한 상승

    일반 사용자가 관리자 권한 획득 가능
